Projects

Along with my journey of learning Web Development, here are some of my projects.

U Rescue Me

A Crowdsourcing Web Application that supports people who wants to reach someone for help to provide information by filling up a rescue form. With the support from U Rescue Me, authorities can easily plan and coordinate their missions. This is a full stack MERN Project collaborated with a team from Chingu IO.

Bangon Pilipinas

Bangon Pilipinas serves as a platform wherein you can easily search and find various donation drives conducted in the Philippines. Built with ReactJS with Algolia API and deployed in Netlify.

Planet Impact Podcast

Planet Impact Podcast is a weekly show on how non-conformist social entrepreneurs are changing the world. This website was built in order to increase engagement and the brand's online presence. Built with HTML, CSS, JS, and deployed in Netlify.

Gulapa Law Redesign

Gulapa Law Redesign keeping their current branding in mind, while improving the overall UX/UI of their website. Built with VueJS, Nuxt, Bootstrap and SCSS. Deployed with Netlify. In collaboration with Daniel Fordan.

World Unite! Redesign (Figma)

This a concept redesign of World Unite's homepage using Figma.

Search Pexel Photos

A simple web application to search stock photos to help creators find the perfect photo for their content using Pexels API. Built with HTML, CSS, and JS. Deployed with Netlify.

Photography Portfolio

This portfolio contains my street photographs from around Singapore. It is customized using Adobe Portfolio with HTML and CSS.
